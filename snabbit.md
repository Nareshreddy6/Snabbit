# Snabbit – Driving Habit Formation in 10-Minute Home Services

## 📍 Problem Understanding

**Context:**

Snabbit offers hyperlocal home services (e.g., cleaning, dishwashing) in under 10 minutes, targeted at busy urban households in Mumbai and Bengaluru.

**Core Problem:**

Despite high trial usage, only **15% of users use Snabbit habitually** (≥2 times/week). This low habit rate limits:

- 🔁 **User retention**
- 💸 **Revenue and LTV**
- 🧱 **Snabbit's ability to become a "default" solution in daily routines**

---

## 🎯 Objective

**Increase the Snab-Habit Rate from 15% → 30% in 6 months**

This will directly impact:

- 🔁 Weekly user retention
- 📈 Customer lifetime value (CLTV)
- 🧱 Product stickiness in household routines

---

## 🔍 Research & Insights

- **Category**: On-demand home services (cleaning, laundry, dishwashing)
- **Target**: Urban professionals in dense micro-markets (Mumbai, Bengaluru)
- **Market Size**: $4B+ urban home services market in India, growing 25–30% CAGR
- **User Profile**: 25–40 yrs, time-poor, tech-savvy, already use Zepto/Swiggy
- **Core Need**: Fast, reliable help when maids are absent or unavailable
- **Snabbit's Edge**: 10-min SLA + full-stack ops (sourcing, training, delivery)
- **Traction**: 25K+ customers, 3 txns/month avg, ₹250–₹270 ticket size
- **Competitors**: Urban Company (slower, premium); informal maids (cheap, unreliable)
- **Opportunity**: High-frequency use, habit loops, micro-market depth, subscriptions
- **Risks**: Trust barrier, ops complexity, user churn if not sticky

## 🔧 Problem Reframing & Retention Funnel

### 🚩 Redefined Problem Statement

Despite strong initial interest and a compelling 10-minute service model, Snabbit fails to convert trial users into recurring ones. Users don't understand what the service can help with, find it costly, and lack emotional or contextual triggers to form a habit.

**This results in a broken loop between first use and habit formation.**

### 🧭 Retention Journey Mapping

| Funnel Stage | % of Users | Drop-Off Reasons |
| --- | --- | --- |
| Awareness | 100% | No major drop-off |
| Onboarding | ~87% | Value unclear, ambiguous task examples |
| First Use | ~60% | Unengaging flow, unsure of what to do next |
| Repeat Use | ~25% | No reminder or hook, perceived cost |
| Habitual Use | ~15% | No habit loop, generic experience |

🗣 **Quotes from Users:**

- ❌ "I signed up but wasn't sure what to do next."
- ❌ "Tried once, didn't feel worth repeating."
- ✅ "Once I used it regularly, it helped reduce my mental load."

## 🧠 Behavioral Patterns & Key Insights

### 🧩 Patterns

| Pattern | Meaning | Supporting Data |
| --- | --- | --- |
| One-time use | Weak first use experience | 33% used Snabbit only once |
| Emotion-triggered use | Opportunity for stress-based prompts | 26% book to relieve stress |
| Reverts to old habits | No behavior switch incentive | 20% say it's backup or unreliable |
| Ambiguity → Inaction | Clarity is missing | 22% unclear about what Snabbit can do |
| Trust friction | Limits repeat usage | 62% want repeat helpers, 15.2% trust issues |

### 🔎 Key Behavioral Biases

| Bias | Meaning | Intervention |
| --- | --- | --- |
| Inertia | Users stick with DIY/full-time help | First Snab free, habit starter pack |
| Forgetting | Users don't think of it | Smart reminders + calendar sync |
| Ambiguity Aversion | Confused users avoid action | Task Gallery + onboarding templates |
| Emotional Triggers | Overwhelm = booking impulse | Contextual nudges ("Feeling drained?") |
| Trust Gap | Inconsistent experience = drop-off | Show helper profiles + rebooking option |

## 📊 Framework & Solution Design

### 📊 ICE Framework

| Initiative | Pain Point | ICE Score | Notes |
| --- | --- | --- | --- |
| **Snab My Week + Smart Reminders** | Forgetting to book | 100 | Easy to build, tackles #1 problem (67% forget) |
| **Task Gallery + Templates** | Unclear use cases | 80 | High impact on onboarding clarity (52%) |
| Loyalty Packs + ₹1 Trial | Pricing concern | 64 | Converts curious users into low-risk trialers |
| Rebook Same Helper | Trust issue | 60 | 71% prefer repeat helpers |
| Emotion Check + Habit Streaks | Emotional flatness | 48 | Adds delight, drives loyalty |
| Waitlist Alerts + Save Time Slot | Availability mismatch | 36 | Mid-impact, moderate build effort |
| Personalized Snab Dashboard | Generic UX | 18 | Useful but high complexity |

---

## 🌟 Selected Initiatives

### ✅ 1. Snab My Week + Smart Reminders

- Auto-booking based on past routines
- WhatsApp & in-app nudges synced to habits
- Highest ICE score, lowest tech barrier

### ✅ 2. Task Gallery + Templates

- Guided booking suggestions
- Onboarding support for new users
- Visual clarity → faster activation

**Why These Ideas Stand Out?**

- They directly address the most common user pain points: forgetting to book and uncertainty about what Snabbit can do.
- Both ideas offer high impact with minimal development effort, making them perfect for early sprint MVPs.
- Backed by real user data, so they're not just guesses — they're validated opportunities.
